About this project.

A CI/CD pipeline built with Docker, GitHub Actions, Amazon ECR and ECS Fargate — every push to main ends up running as a live container, automatically.

The five stages

  1. 01

    Push

    Code lands on main in GitHub and triggers the workflow.

  2. 02

    Build & test

    GitHub Actions installs dependencies, runs checks, and prepares the build.

  3. 03

    Containerize

    Docker builds an image of the app from a multi-stage Dockerfile.

  4. 04

    Publish

    The image is tagged and pushed to a private Amazon ECR repository.

  5. 05

    Deploy

    ECS rolls out a new Fargate task from the updated image — no servers to provision or patch.
